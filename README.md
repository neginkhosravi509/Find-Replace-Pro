FindReplacePro is a Windows Forms application designed for searching and managing text inside a textbox.  
The program allows users to find specific words, detect repeated words, and replace selected text quickly and easily.  
It provides a simple and efficient way to handle long or complex text through a clean and user-friendly interface.

Features
- Search for any word in the text  
- Detect repeated words  
- Highlight and locate found words  
- Replace selected words with new text  
- Simple and intuitive Windows Forms design  

FindReplacePro is a practical tool for editing, reviewing, and managing text with ease.
